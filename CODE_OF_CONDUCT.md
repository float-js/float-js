# Code of Conduct

## Our Standards

This project maintains a professional environment. Expected behavior includes:

- Using clear and respectful language
- Accepting constructive feedback
- Focusing on technical merit
- Respecting differing viewpoints
- Maintaining professional conduct

Unacceptable behavior includes:

- Harassment or discriminatory language
- Personal attacks or insults
- Trolling or inflammatory comments
- Publishing private information without consent
- Other unprofessional conduct

## Enforcement

Violations may result in temporary or permanent restrictions from project participation. Reports can be made to project maintainers via GitHub or email.

## Scope

This Code of Conduct applies to all project spaces including GitHub repositories, discussions, and any official communication channels.
